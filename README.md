# Advent of Code 2025 – Solutions en Python

<https://adventofcode.com/2025/>

Ce dépôt contient mes solutions personnelles à **Advent of Code 2025**, écrites en **Python**.

L’objectif n’est pas seulement de “trouver la bonne réponse”, mais aussi de travailler :

- la modélisation d’un énoncé en code
- l’écriture Python propre et lisible
- le raisonnement algorithmique
- la gestion de cas particuliers
